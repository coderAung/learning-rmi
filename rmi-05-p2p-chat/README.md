## Simple P2P chat program with RMI

This Peer 2 Peer chat program is a minimum example of using RMI.

### Usage guid

```
# clone git repo
# javac src\dev\aung\rmi\chat\Main.java
# java dev.aung.rmi.chat.Main
```

The program will ask you to enter a name for unique id of node.
Then to connect with other nodes if they are in connection,
we need to enter the node's name to connect.
Then simple chatting begin.

Program provides you to exit with /exit keyword.
